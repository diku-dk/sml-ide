[parent e]
returns SOME p, if p is the parent of e. Returns NONE if e has no parent.

[appendChild e child]
appends child to e.

[removeChild e child]
removes child from e.

[replaceChild e new old]
replaces old child from e with new child.

[random()]
returns a random real in the interval [0.0,1.0[.

[loadScript url callback]
loads the JavaScript file specified by the url and execute the callback function once the script is fully loaded. The loadScript function assumes that a head element is present in the DOM.
